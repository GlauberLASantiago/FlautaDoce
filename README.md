# Flauta Doce Interativa - Profa. Isamara Carvalho

Este é um aplicativo web interativo projetado para o ensino e prática da **Flauta Doce** (Soprano, Contralto e Tenor). O projeto foca em acessibilidade e gamificação, permitindo que alunos explorem dedilhados de forma visual e testem seus conhecimentos através de um módulo de treinamento com feedback sonoro em tempo real.

## 🚀 Funcionalidades

- **Múltiplos Instrumentos:** Suporte para Flauta Soprano, Contralto (F) e Tenor.
- **Modo Livre:** Exploração visual de dedilhados com reprodução de áudio de alta fidelidade.
- **Modo Treino:** 6 níveis de dificuldade progressiva, desde notas diatônicas básicas até escalas cromáticas completas e notas agudas.
- **Sistema de Áudio Avançado:** - Uso de SoundFonts para timbres realistas.
    - Controle de **Reverb** dinâmico para simulação de ambiente.
    - Efeitos sonoros diferenciados para acertos (Xilofone) e erros (Timbal).
- **Interface Responsiva:** Otimizado para dispositivos móveis (touch-friendly) e desktops.
- **Design Moderno:** Estética inspirada em sintetizadores modernos com feedback visual por luzes e animações.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
- **Áudio:** [soundfont-player](https://github.com/danigb/soundfont-player) para síntese baseada em samples.
- **Web Audio API:** Para processamento de áudio, ganhos e convolução de reverb.
- **Fontes:** Google Fonts (Anton e Montserrat).

## 📖 Como Usar

1. **Seleção de Instrumento:** Use o menu superior para escolher o tipo de flauta.
2. **Modo Livre:** Clique ou toque nos diagramas de flauta para ouvir a nota e ver o dedilhado correspondente.
3. **Modo Treino:** - Clique em "Treino" e escolha um nível.
   - O app exibirá o nome de uma nota e você deve tocar no diagrama correto antes que o tempo se esgote.
   - O sistema contabiliza acertos e erros, fornecendo uma porcentagem de aproveitamento ao final de 10 rodadas.

## 📂 Estrutura do Projeto

O projeto é entregue como um **Single File Application (SFA)**, contendo todo o HTML, CSS e JavaScript em um único arquivo para facilitar a distribuição e o carregamento offline em ambientes virtuais de aprendizagem como o **Moodle**.
